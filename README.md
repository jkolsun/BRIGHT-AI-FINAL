# Bright.AI - Landscaping Management Platform

## Description
Bright.AI is a comprehensive landscaping business management platform with separate interfaces for administrators and crew members.

## Features
- **Admin Dashboard**: Manage jobs, quotes, crews, and messages
- **Crew Mobile App**: Clock in/out, view jobs, track progress
- **Smart Import**: Import data from CSV or Google Sheets
- **Live Tracking**: Real-time crew and job location tracking
- **Database Integration**: Connected to Supabase for real-time data sync
